# Mini Life Simulator

A text-based life simulator written in Python.

## Features
- Health, Energy, Happiness, IQ, and Money stats
- Random daily events
- Rare and legendary events
- Milestone events (Day 25, 50, 100)
- Aging system
- Immortality mechanic
- Multiple endings

## How to Run

1. Make sure Python is installed.
2. Download the file.
3. Run the program:

python life_simulator.py
